##Biblioteca
***
El archivo biblioteca.zip contiene 2 carpetas y un archivo.

Apibiblioteca: Esta carpeta contiene la aplicacion en laravel que funciona como servidor del proyecto.
Una forma de utilizar esta tecnologia es con el software de Composer(https://getcomposer.org/), en esta pagina
podras encontrar los detalles de instalacion.

Una vez que este instalado debes lanzar el siguiente comando en CMD en la carpeta que descomprimas el proyecto.
php artisan serve <--- Para correr el sistema

Para que el proyecto funcione sin problemas deberÃ¡s haber importado la BD que se adjunta como biblioteca.sql en el zip
principal. 

La otra carpeta contiene un proyecto desarrollado en angular, que fungira como frontend de la aplicacion.
Para su funcionamiento basta con abrir el archivo en la linea de comandos e ingresar los comandos:
npm i <--------- para que se instalen todos los modulos necesarios para el proyecto
npm start <----- para correr el proyecto
